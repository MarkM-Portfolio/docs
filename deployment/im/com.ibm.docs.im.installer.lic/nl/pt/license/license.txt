"Proporção n/m" significa que o Licenciado recebe algum número ("n") de direitos para o Programa de Suporte para cada número ("m") especificado de direitos do Programa Principal como um todo. Salvo especificação contrária, o número de direitos do Programa de Suporte é arredondado para cima, até um múltiplo de "n". Por exemplo, se um programa inclui 100 PVUs para um Programa de Suporte para cada 500 PVUs obtidas do Programa Principal e o Licenciado adquirir 1.200 PVUs do Programa, o Licenciado poderá instalar o Programa de Suporte e ter núcleos de processador disponíveis para ele ou gerenciados por ele até o máximo de 300 PVUs. Essas PVUs não precisam ser contadas como parte do requisito total de PVUs para a instalação do Programa pelo Licenciado por conta da instalação do Programa de Suporte (embora possa ser necessário contar essas PVUs por outros motivos, como núcleos de processador sendo disponibilizados para o Programa Principal, também).

"Uso pelo Programa Principal" significa que o Programa de Suporte é fornecido exclusivamente para uso pelo Programa Principal. Nem o Licenciado nem qualquer aplicativo/dispositivo programador está autorizado a usar ou acessar diretamente os serviços do Programa de Suporte, com exceção do Licenciado, que tem permissão para acessar o Programa de Suporte para executar funções administrativas, como backup, recuperação e configuração autorizada.
