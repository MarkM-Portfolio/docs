1. Goal

Package  SpellCheckService.class into com.ibm.docs.spellcheck.service.jar

Generally SpellCheckService.class is not changed frequently. So this build process is not put the build process of Docs.

2. How to build com.ibm.docs.spellcheck.service?

2.1 ant -f {location of project com.ibm.docs.spellcheck.service}/build.xml

2.2 find com.ibm.docs.spellcheck.service.jar under com.ibm.docs.spellcheck.service\bin\jar\

2.3 put latest jar into com.ibm.docs.spellcheck/lib
