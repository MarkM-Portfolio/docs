豪华旗舰逆生长 奔驰GLS将3月10日上市

2016-02-18 10:05:11　来源: 网易汽车
0

奔驰GLS为现款GL的中期改款车型，新车将更名为奔驰GLS，将于3月10日正式上市。与现款车型相比，新车的外观内饰设计方面都更时尚和年轻化。

版权声明：本文版权为网易汽车所有，转载请注明出处。

网易汽车2月18日报道 日前有消息表示，新款奔驰GLS将于3月10日正式上市。作为中期改款车型，新车采用了奔驰最新的命名方式，以及更加年轻时尚的设计风格。

信息点：1.整体年轻化 2.动力提升/标配9AT 3.全新人机交互系统

新车信息：

·现款奔驰GL的中期改款车型，新车更名为奔驰GLS；

·外观上，主要针对家族脸进行了调整，线条更圆润时尚；

·大灯升级采用了LED光源，LED日间行车灯则整合到了大灯内部；

·尾部灯组换装LED光源，还配备了镀铬底盘护板；

·内饰采用菱形纹路的皮革和木质装饰材料的结合；

·配备三辐式多功能方向盘、可变色的氛围灯、8寸液晶显示屏；

·全新的人机交互系统增加了在线互联服务：包含紧急呼叫、事故救援、车辆诊断等功能。

    两款动力选择：

    ·3.0T V6双涡轮增压发动机：最大功率270千瓦，峰值扭矩500牛·米；

    ·4.0T V8双涡轮增压发动机：最大功率339千瓦，峰值扭矩700牛·米。

    编辑点评：

    根据此前消息，GLS级还将拥有Coupe（GLS运动SUV）、长轴版以及迈巴赫GLS级等更多衍生车型，作为公路型SUV当之无愧的旗舰，全新GLS级将从多个方面压榨对手。而现款GL级甚至还存在着加价现象，希望GLS级上市后能在售价方面给我们一个惊喜。
