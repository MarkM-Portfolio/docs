Αγαπητέ κ. Μανωλίδη,


Είμαι στην ευχάριστη θέση να σας παρουσιάσω το νέο πακέτο εργαλείων παραγωγικότητας HCL Docs.


Το HCL Docs είναι ένα πακέτο διαδικτυακών εργαλείων για τη δημιουργία και την κοινή χρήση και επεξεργασία εγγράφων κειμένου, υπολογιστικών φύλλων και παρουσιάσεων. Τα εργαλεία αυτά είναι ιδιαίτερα χρήσιμα αφού σας επιτρέπουν να αναπτύσσετε περιεχόμενο από κοινού με τους συνεργάτες σας, να σχολιάζετε και να συζητάτε σχετικά με τα έγγραφα που επεξεργάζεστε, και να αναθέτετε στα μέλη της ομάδας σας την εκτέλεση εργασιών για συγκεκριμένες ενότητες των εγγράφων σας.


Αν χρειάζεστε περισσότερες πληροφορίες, μην διστάσετε να επικοινωνήσετε μαζί μου.
