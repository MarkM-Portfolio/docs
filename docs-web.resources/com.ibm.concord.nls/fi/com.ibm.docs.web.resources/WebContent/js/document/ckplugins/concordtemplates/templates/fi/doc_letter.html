Arvoisa toimitusjohtaja Babineaux


Minulla on ilo esitellä teille uusin tuottavuustyökalu, HCL Docs.


HCL Docs on Web-perustainen muokkausohjelma, jolla voi luoda, määrittää yhteiskäyttöön ja laatia yhdessä asiakirjoja, laskentataulukoita ja esityksiä. Asiakirjojen luominen on entistä kätevämpää, koska voitte yhdessä työtovereiden kanssa muokata ja kommentoida sisältöä ja keskustella siitä sekä määrittää eri osia työryhmän jäsenten tehtäväksi.


Odotan yhteydenottoanne. Vastaan mielelläni mahdollisiin kysymyksiinne.
