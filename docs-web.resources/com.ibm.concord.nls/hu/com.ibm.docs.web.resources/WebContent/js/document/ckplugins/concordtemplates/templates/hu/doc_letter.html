Engem ért az a megtiszteltetés, hogy ebben az e-mailben bemutathassam Önnek legújabb, HCL Docs nevű termelékenységsegítő eszközünket.


Az HCL Docs egy webalapú szerkesztőprogram, amely dokumentumok, táblázatok és bemutatók készítésére, megosztására és több munkatárs által közösen történő szerkesztésére szolgál. Jobb megoldást nyújt az eddigieknél a dokumentumok készítésére, mert több munkatárs működhet együtt a szerkesztés során, tehet megjegyzéseket és vitathatja meg a dokumentum tartalmát a környezettől függően, és a dokumentum egyes részeit ki lehet osztani a csoport egyes tagjai számára.


Ha bármilyen további információra lenne szüksége, készséggel állok rendelkezésére.
